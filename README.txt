To launch the bot:
> Extract this folder to your desktop
> Open VS Code or PowerShell
> Navigate to bot Folder using the following commands if you 
arent already there:
	cd destination    (go to destination folder)
	cd ../		  (go up one folder)
	ls		  (list all files in folder)

> Type in 'npm build' and run it
> Once finished you can type in 'node bot.js' to run the bot


] To shut it down 'Ctrl + C' in the terminal
] if that doesnt work, tell the bot 'shutdown' through discord
